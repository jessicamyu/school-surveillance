Reallocation of Capital Class Size Reduction Funds
Meeting
Aug 6, 2008 - Spec Mtg re Class Size Reduction Funds
Category
Consent Agenda
Type
Action


  1. This action allocates the residual balance of Classrooms First funding to a current Class Size Reduction project.


  2. The Department of Education requires School Board approval of any re-allocations of these funds before they will authorize any adjustments.


  3. The projects originally approved for these funds were completed under budget resulting in a balance of $233,284.78. 


  4. Those funds will be applied to a project that has received other Class Size Reduction funds, specifically the Pine Jog Environmental Elementary School.  An equal amount of local funding will be transferred from the project to the Capital Contingency account so there will be no change to the project budget. 


 

Recommendation:

 I recommend the School Board approve a revised encumbrance authorization for the Classrooms First funds and authorize the Superintendent and the School Board Chairman to execute all necessary documents to access State funding.
